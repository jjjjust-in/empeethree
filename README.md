# Empeethree 🎵

A terminal-aesthetic desktop MP3 player built with Electron.

## Setup

```bash
# 1. Install dependencies
npm install

# 2. Run the app
npm start
```

## Features

- **Playlist / Queue** — load individual files or an entire folder
- **Waveform Visualizer** — real-time waveform + frequency bars via Web Audio API
- **Transport Controls** — play, pause, previous, next
- **Shuffle** — randomized playback
- **Repeat** — off / repeat all / repeat one
- **Seek Bar** — click or drag to scrub
- **Volume Control** — click or drag to adjust
- **Terminal Aesthetic** — JetBrains Mono, amber/green accents, scanlines

## Keyboard Shortcuts

| Key         | Action              |
|-------------|---------------------|
| `Space`     | Play / Pause        |
| `N`         | Next track          |
| `P`         | Previous track      |
| `→`         | Seek forward 5s     |
| `←`         | Seek back 5s        |
| `↑`         | Volume up           |
| `↓`         | Volume down         |

## Supported Formats

MP3, WAV, OGG, FLAC, M4A
